## Reverse A Number

### Easy
***

1. You've to display the digits of a number in reverse.
2. Take as input "n", the number for which digits have to be display in reverse.
3. Print the digits of the number line-wise, but in reverse order.

**Example 1:**
<pre>
<b>Input: </b> 65784383
<b>Output: </b>
3
8
3
4
8
7
5
6
</pre>

**Constraints:**
```
1 <= n < 10^9
```
