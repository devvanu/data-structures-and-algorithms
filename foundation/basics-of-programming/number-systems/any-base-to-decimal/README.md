## Any Base To Decimal

### Easy
***

1. You are given a number n.
2. You are given a base b. n is a number on base b.
3. You are required to convert the number n into its corresponding value in decimal number system.

**Example 1:**
<pre>
<b>Input: </b> 111001 2
<b>Output: </b> 57
</pre>

**Example 2:**
<pre>
<b>Input: </b> 1266 8
<b>Output: </b> 694
</pre>

**Constraints:**
```
0 <= d <= 1000000000
2 <= b <= 10
```
