## Count Digits In A Number

### Easy
***

1. You've to count the number of digits in a number.
2. Take as input "n", the number for which the digits has to be counted.
3. Print the digits in that number.

**Sample Input:**
```
65784383
```

**Sample Output:**
```
8
```

**Constraints:**
```
1 <= n < 10^9
```
