## Toggle Case

### Easy
***

1. You are given a string that contains only lowercase and uppercase alphabets. 
2. You have to toggle the case of every character of the given string.

**Example 1:**
<pre>
<b>Input: </b> pepCODinG
<b>Output: </b> PEPcodINg
</pre>

**Constraints:**
```
1 <= length of string <= 1000
```
