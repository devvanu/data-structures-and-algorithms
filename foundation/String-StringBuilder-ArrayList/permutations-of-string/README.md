## Print All Permutations Of A String Iteratively

### Easy
***

1. You are given a string. 
2. You have to print all permutations of the given string iteratively.

**Example 1:**
<pre>
<b>Input: </b> abc
<b>Output: </b> 
abc
bac
cab
acb
bca
cba
</pre>

**Constraints:**
```
1 <= length of string <= 15
```
