## Digits Of A Number

### Easy
***

1. You've to display the digits of a number.
2. Take as input "n", the number for which digits have to be displayed.
3. Print the digits of the number line-wise.

**Example 1:**
<pre>
<b>Input: </b> 65784383
<b>Output: </b>
6
5
7
8
4
3
8
3
</pre>

**Example 2 (Special Case):**
<pre>
<b>Input: </b> 1554538760
<b>Output: </b>
1
5
5
4
5
3
8
7
6
0
</pre>

**Constraints:**
```
1 <= n < 10^9
```
