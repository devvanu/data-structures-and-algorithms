## Decimal To Any Base

### Easy
***

1. You are given a decimal number n.
2. You are given a base b.
3. You are required to convert the number n into its corresponding value in base b.

**Example 1:**
<pre>
<b>Input: </b> 57 2
<b>Output: </b> 111001
</pre>

**Example 2:**
<pre>
<b>Input: </b> 694 8
<b>Output: </b> 1266
</pre>
