## Pythagorean Triplet

### Easy
***

1. You are required to check if a given set of numbers is a valid pythagorean triplet.
2. Take as input three numbers a, b and c.
3. Print true if they can form a pythagorean triplet and false otherwise.

**Example 1:**
<pre>
<b>Input: </b> 5 3 4
<b>Output: </b> true
</pre>

**Constraints:**
```
1 <= a <= 10^9
1 <= b <= 10^9
1 <= c <= 10^9
```
